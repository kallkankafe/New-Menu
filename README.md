# Rrem's Café & Hookah Website

Modern one-page website for Rrem's Café & Hookah in Tetovo.

## Before publishing

1. Update the real WhatsApp number in `js/main.js`:
   ```js
   const WHATSAPP_NUMBER = "38970000000";
   ```
2. Replace `https://rrems.mk/` with the real domain in:
   - `index.html`
   - `robots.txt`
   - `sitemap.xml`
3. Add the real address, phone number, Instagram URL and opening hours inside the JSON-LD block in `index.html`.
4. Upload all files to the public hosting root.
5. Add the domain to Google Search Console and submit `sitemap.xml`.

## SEO included

- Clean title and meta description
- Canonical URL
- Open Graph and Twitter preview tags
- LocalBusiness / CafeOrCoffeeShop JSON-LD structured data
- Semantic headings and image alt text
- Static menu HTML for better crawlability
- `robots.txt`
- `sitemap.xml`

## Design direction

- Brand colors: dark navy `#0F1822`, gold `#CD9D2E`, warm cream
- Brand-style font direction with Cinzel for logo/headings and Manrope for UI
- Non-template circular hero composition
- Smoke, shine and particle effects using CSS
- Less text, more atmosphere
